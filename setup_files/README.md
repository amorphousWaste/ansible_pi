# Setup Files #
Files that are copied over as part of the deployment process.

## .bashrc ##
Basic `.bashrc` file to use in the `pi` profile.

## apache2.conf ##
Apache configuration modified with some "hardening" to prevent common attacks.

## ssl_key_example ##
An example key to copy over. This should be replaced with a real, signed one.

## retropie.md ##
A text file with some hints on how to set some things up.

## panel ##
A PIXEL panel configuration file for the Raspberry Pi desktop.

## raspi-config.sh ##
This file can be modified to run tasks either on the next boot or on every boot.

## Kernel Docs ##
Some good info can be found about the kernel and supporting overlays at `/boot/overlays/README`
